# Release Notes Draft

## 1.0.0 internal test

Initial internal test build for Maxxed Gold Estimator.

- Guided six-angle sample capture.
- Known-size reference calibration.
- On-device visual material clustering.
- User review for material labels and mask sensitivity.
- Field sample context for panning, sluice cleanup, paydirt, tailings, and hard-rock specimens.
- Optional total sample weight, gold price per gram, and expected recovery percentage.
- Recoverable Gold weight and rough field value ranges.
- Saved batches, comparison, copy summary, and CSV export.

All results are visual estimates only. This app does not perform a chemical assay, purity test, appraisal, live market price lookup, or guaranteed identification.

# Store Listing Draft

## App name

Maxxed Gold Estimator

## Short description

Offline multi-angle visual estimates for mixed field samples.

## Full description

Capture a sample from six guided angles, measure a known-size reference, and review an entirely on-device visual analysis. Correct the subject mask and material labels, then view conservative ranges for visible share, density, volume, weight, and confidence.

Save and compare batches, copy summaries, or explicitly export CSV. The app uses no account, cloud analysis, ads, or analytics.

Maxxed Gold Estimator provides visual estimates only. It does not determine chemical composition and is not a laboratory assay. Gold-colored material may be pyrite, mica, staining, glare, or another look-alike. Use qualified testing for decisions involving identification or value.
